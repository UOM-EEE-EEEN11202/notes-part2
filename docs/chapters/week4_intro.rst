Week 4
======
The aim of Week 4 is to introduce some common external Python libraries which are widely used in Electronic Engineering. There are many thousands of external libraries available for Python, and we won't cover them all. We'll focus on a few key libraries which are widely used in engineering applications. This will help build some familaritity and move us towards more real-world applications of Python.

In the latter parts of :ref:`Lab F <lab_f_stage_2>` we'll also look at a handful of more advanced Python concepts, mainly splitting code into multiple files, to help us as our code gets bigger.
